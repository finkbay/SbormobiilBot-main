ИНСТРУКЦИЯ

БОТ
1. В директории \SbormobilBot\tgadmin создать файл .env и записать в него токен бота в виде: TOKEN="СЮДА НАПИСАТЬ ТОКЕН БОТА(без кавычек)"




АДМИНКА
1 В командной строке сменить рабочий каталог, на каталог бота SbormobilBot. Пример(" cd G:\SbormobilBot ", после чего сменить том:  " G:  ")
2 Запустить сервер с помощью команды " python tgadmin\manage.py runserver "
3 перейти по написанному адрессу дописав после него admin (  http://127.0.0.1:8000/admin  
4 Залогиниться
5 Нажав на Отчётs, откроется меню, где можно будет удалять и добавлять строки отчёта
